    class HttpProxy()¶

    实例¶

    本实例利用HttpProxy实现了一个将所有响应修改为在页面顶部中央显示It works!的代理

    from utils import *
import net
import time

p=net.HttpProxy("127.0.0.1:2077") # 监听本机2077端口
p.start() # 启用代理
while True:

    req=p.request() # 获取待处理请求
    if req!=None: # 如果取得请求
        req.release() # 释放请求

    resp=p.response() # 获取待处理响应
    if resp!=None: # 如果取得响应
        resp.headers({
            "Content-Type":"text/html"
        }) # 修改响应头
        resp.body("<html><center>It works!</center></html>") # 修改响应体
        resp.release() # 释放响应

    time.sleep(0.1) # 记得使用sleep避免cpu占用过高
